# Backups

Folder ini hanya untuk arsip lokal dan hasil pemulihan.

## Isi Saat Ini

- `pending_updates_backup/` berisi snapshot JSON hasil backup update yang lama.

## Aturan

- Jangan taruh file kerja baru di root repo.
- Jika membuat backup baru, letakkan di subfolder bertimestamp.
- Backup besar tidak dimaksudkan untuk masuk git.
