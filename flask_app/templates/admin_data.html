Tambah Channel Baru

Setelah channel ditambahkan, pipeline discovery → transcript → resume → format akan dijalankan otomatis di background.

Tambah Multiple Channel

Setiap channel yang berhasil ditambahkan akan langsung dijadwalkan pipeline discovery → transcript → resume → format secara background.

Discovery - Cek Video Baru

Scan channel untuk menemukan video baru yang belum ada di DB. Tanpa download transcript.

Download Transcript

Download transcript untuk video yang belum ada transkripnya (bukan no_subtitle).

Resume Generation (AI)

Buat ringkasan video menggunakan model AI (Nvidia/Groq/dll). Membutuhkan transkrip yang sudah terunduh.

Update Metadata Video

Update duration, view count, like count, comment count, dan thumbnail video dari YouTube Data API.

Markdown Formatting

Rapikan transkrip mentah menjadi format Markdown yang cantik. Membutuhkan transkrip yang sudah terunduh.

Hapus Channel

Hapus channel beserta seluruh video dari database. File fisik di uploads TIDAK dihapus.

Jalankan Pipeline Berurutan (Sequential)

Jalankan seluruh pipeline secara berurutan (sequential):
1. Discovery → 2. Transcript → 3. Resume → 4. Format
Setiap stage akan dijalankan setelah stage sebelumnya selesai.
Catatan: Resume & Format pakai resource key yang sama (rate limit), jadi jangan jalankan bareng-bareng.
